光谷K11购物艺术中心开业引流项目

光谷K11购物艺术中心，坐落于武汉光谷核心，定位“时髦、雅致、轻奢”，打造“云、湖泊、瀑布”科技互动装置，革新性的带来5.4万方“博物馆零售”新体验。

作为华中首个购物艺术中心，馆内外还分布有九件由国内外艺术家创作的艺术典藏，供顾客近距离观赏体验。2017年11月2日，光谷K11，以“艺术主义”、“潮流先锋”、“未来生活”为核心，开启全新创艺生活体验之旅。

该项目结合K11各大入驻品牌和艺术典藏(艺术装置)，通过微信公众号二级菜单进入SocialPoint系统和两款互动游戏完成互动任务，并结合线下互动屏进行互动获取Social Point(互动积分)，通过Social Point可以兑换丰富好礼，实现成功引流。

产品概述Product Overview

K11《B》感恩之旅是一个集互动屏+crm+H5联动的互动方案。通过线下大屏玩游戏互动，奖励Social Point(互动积分)，能够推广关注K11公众号并吸引用户到武汉K11线下消费。该方案是此次开业活动中的重要一环。
用户在K11商场经过熊一体机时，熊讲述故事并且需要用户帮助（收集爱心），用户帮助熊获得爱心，并且开始游戏（闯关）。通过闯关获得足够爱心，游戏结束。与熊合影，并且微信扫描二维码，进入H5，用户获得Social Point(互动积分)。

需求要点Requirements Document

1. 硬件：手机、互动屏、Kinect体感控制器
2. 核心元素：艺术装置的廖国核-B，艺术品“熊”有关联（不可直接使用此熊形象）
3. UI风格：L2楼层是以儿童及文创类品牌为主，此互动UI呈现方面需要与K11品牌调性匹配。
4. 交互场景：以儿童亲子体验为主，在大屏进行简单易操作、短时高效的互动体验。
5. 分享功能：拍照，父母喜欢让孩子与可爱的动物、植物等场景下拍照。
6. 排名功能：可以通过游戏成绩的高低刺激用户反复参与进来，并根据游戏成绩获得互动积分。
7. 玩法难易度：分为“低中高”，增强游戏者的挑战性和趣味度。

说明：

1. 关于Social Point的运用，K11 Central团队 & CRM团队会提供参考值，需要考虑通过CRM+系统的消息通知模板运用，提升用户的互动粘性或者交叉销售。
2. 因为此互动点位处于电梯厅出来“人流侧向位置”，需要考虑是否增加“感应”触碰，引发经过人群的关注来参与互动？例如可以考虑考虑结合“微信扫封面方式”引导客人扫描艺术品进行交互，引导来到触摸屏做互动.

产品概述Product Overview

K11“鱼”你在一起也是一个集互动屏+crm+H5联动的互动方案。游戏主要讲述鱼与魔法的故事。玩家通过体感控制海底世界的鱼，通过重重关卡穿过魔法之书，进入魔法世界。通关微信扫描二维码，进入H5页面，获取Social Point(互动积分)和神秘拼图碎片。

需求要点Requirements Document

1. 硬件：手机、互动屏、Kinect体感控制器。
2. 核心元素：考虑与湖泊装置有关联的故事场景，不太建议很复杂的交互。

4. 具有年轻元素：可考虑加入运势、星座、贴标签等元素增加年轻人分享率。
5. 增加互动感：互动内容可考虑发生在mall内的2人以上（如情侣、闺蜜、朋友等）。
6.在用户选择的过程中可以考虑埋入消费者行为tagging元素，如购买兴趣、活动兴趣等。

产品概述Product Overview

传统的线下植物种植的四大限制：种植面积有限，植物类型有限，种植周期较长，可互动频次较低，参与人数有限。可否突破上述限制，让从土地里“转化”到线上移动端，再由线上移动端“转化”到土地里？
在都市农庄里开展线下种植课程，客人自带植物参与活动，并将客人线下种植的植物“传送”至线上，引导参与线上互动，通过线上渠道维护客人种植后的粘性，定期唤醒，同时嫁接各项产品，成熟之后曲线下农庄将有机会领取惊喜奖品。

需求要点Requirements Document

1. ui风格：以亲子体验为主，风格跟其他项目保持大体一致，呈现方面需要与K11品牌调性匹配。
2. 交互场景：手机H5虚拟种植植物，与孩子一起学习植物种植小知识。
3. 营养包：用Social Point在线上H5中购买营养包加速成长。
4. CRM+系统：根据其过往记录，推送其返场当天适合的信息（优惠券、K11活动等），引导其消费。
